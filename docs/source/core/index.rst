Core Analysis (``peccary.peccary``)

Introduction
============

The ``peccary`` module is the heart of the the ``peccary`` package
and is used for calculating the ordinal patterns, probability distributions,
Permutation Entropy, and Jensen-Shannon Statistical Complexity.
